## 上传方式

1. 打开：https://github.com/double4q/AImbti
2. 点击 `uploading an existing file` 或 `Add file` → `Upload files`
3. 上传：
   - `README.md`
   - `public/ai-user-personality-test-prompt.md`
4. Commit message 可以写：
   - `Add public AI personality test prompt`
5. 点击 `Commit changes`
