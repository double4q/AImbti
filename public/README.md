# AI 使用者人格测试

这是「AI 使用者人格测试」的公开 Prompt 页面。

## 给用户复制的短启动词

```text
请打开并阅读这个链接里的完整规则，然后严格按照它执行「AI 使用者人格测试」：
https://raw.githubusercontent.com/double4q/AImbti/main/public/ai-user-personality-test-prompt.md

现在请先用开场白开始，不要一次性问完 10 个问题。
```

## Prompt 文件

- [ai-user-personality-test-prompt.md](./ai-user-personality-test-prompt.md)

## 使用说明

1. 用户复制上面的短启动词。
2. 把链接替换成真实公开链接。
3. 发给支持读取网页链接的 AI 模型。
4. 模型读取完整规则后，开始一题一题进行测试。
